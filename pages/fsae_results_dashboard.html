How to use the FSAE Results Dashboard

Choose the field

Competition and year filters control the results included everywhere. All available data is selected when the dashboard first loads.

Build a plot

Select result metrics for X and Y to create competition-result cross-plots, with an optional metric controlling point size. The X menu also provides School Timeseries for tracking a selected school’s metric across years and Competition Timeseries for plotting competition-level metrics over time. Hover or tap a point for its team and values. Overlay Competitions combines events on one plot; turn it off to separate them.

Work with the table

Search for a team, select a heading to sort, and use Columns to add, remove, or reorder fields. Clearing a Plot checkbox hides only that point and leaves its table row in place. Maximize Table uses the full workspace to the right of the filters.

Handle missing and zero results

Hide Zero Values treats individual zero metric values as unavailable. Hide Zero/Zero Values removes a row only when both plotted values are zero. Hide No Weight Entries excludes teams without a recorded vehicle weight.

Run a simulated result

Enable Simulated Results, choose a competition and year, then enter static scores and dynamic-event inputs. All First fills the best observed reference values. The starred team is rescored with the field using 2026 formulas as a historical approximation.

Official competition PDFs remain authoritative. Use Report Data Error if a dashboard value appears incorrect.

FSAE results data disclaimer

OpenDyna results are transcribed and processed from official competition documents and may contain small errors or discrepancies.

The official released PDFs remain authoritative. If you identify an issue, please submit a data correction ticket so it can be reviewed.

Table columns

Use the arrows to change the left-to-right order. The Plot column remains fixed.
